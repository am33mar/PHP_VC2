## Delete branches locally:

git branch -d dev
git branch -d test

## Delete branches remotely:

git push origin :dev
git push origin :test

## checkout another branch without commit changes
git stash
git checkout <the_other_branch>

## List all tags:
git tag

## Delete tag locally:
git tag -d v1.7

## Delete tag remotely:
git push origin --delete v1.7
